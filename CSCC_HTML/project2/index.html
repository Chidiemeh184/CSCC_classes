Difference between Absolute and Relative Path

The difference, simply put, in my own understanding, is that Absolute path will get the full url which will include http:// and the domain name and abosolute path will link to a page on another website or external website whereas relative link will point to the pages in different folder location (or file path) in your website
For example using Absolute link to point to Jenni's flavour page will be https://jenis.com/flavors/ in the a tag Click here
An example of the relative link: In the project 2 folder all the five pages for the project are in the same folder called project 2.
